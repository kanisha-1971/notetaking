# Note-Taking Website (Dark Theme)

This is a simple note-taking web application built using HTML, CSS, and JavaScript. The app features a sleek dark theme and allows users to create and view notes dynamically.

## Features

- **Dark Theme**: The app uses a dark color scheme to reduce eye strain and provide a modern aesthetic.
- **Responsive Design**: Optimized for different screen sizes, making it accessible on both desktop and mobile devices.
- **Note Management**: Users can add notes, which are displayed in a list format.
